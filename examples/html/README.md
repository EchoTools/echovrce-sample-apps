# HTML Examples

This directory contains HTML/JavaScript examples demonstrating API usage.

## Getting Started

1. Open any `.html` file in a web browser
2. Follow the instructions in the example

## Adding Examples

To add a new example:

1. Create a new HTML file with a descriptive name (e.g., `basic-request.html`)
2. Include inline JavaScript or reference external JS files
3. Add comments explaining what the example demonstrates
