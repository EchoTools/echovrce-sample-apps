# Python Examples

This directory contains Python examples demonstrating API usage.

## Getting Started

1. Install Python 3 from https://www.python.org/
2. Install dependencies: `pip install -r requirements.txt` (if applicable)
3. Run an example: `python <example>.py`

## Adding Examples

To add a new example:

1. Create a new `.py` file with a descriptive name
2. Include comments explaining what the example demonstrates
3. Ensure the example is self-contained and runnable
