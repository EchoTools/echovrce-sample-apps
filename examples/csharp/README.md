# C# Examples

This directory contains C# examples demonstrating API usage.

## Getting Started

1. Install .NET SDK from https://dotnet.microsoft.com/
2. Navigate to an example directory: `cd <example-directory>`
3. Run the example: `dotnet run`

## Adding Examples

To add a new example:

1. Create a new console application: `dotnet new console -n <example-name>`
2. Navigate into the project directory
3. Add your API usage code to `Program.cs`
4. Include comments explaining what the example demonstrates
