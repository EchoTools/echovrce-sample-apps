# C# Examples

This directory contains C# examples demonstrating API usage.

## Getting Started

1. Install .NET SDK from https://dotnet.microsoft.com/
2. Run an example: `dotnet run --project <example>.csproj`

## Adding Examples

To add a new example:

1. Create a new project or `.cs` file with a descriptive name
2. Include comments explaining what the example demonstrates
3. Ensure the example is self-contained and runnable
