# echovrce-api-examples

A collection of API usage examples in various programming languages.

## Structure

```
examples/
├── html/       # HTML/JavaScript examples
├── go/         # Go examples
├── python/     # Python examples
└── csharp/     # C# examples
```

## Getting Started

Navigate to the language directory of your choice and follow the README instructions.

## Contributing

To add a new example:

1. Choose the appropriate language directory in `examples/`
2. Follow the guidelines in that directory's README
3. Submit a pull request with your example
